
URL 的理解
```
URL 是什么呢？
Uniform Resource Locator： 统一资源定位符，它表示网络上某一资源的地址

它是干什么的呢？
表示某一资源的地址。

比如：https://blog.csdn.net/lu202032?spm=1018.2226.3001.5343
协议：https
主机名，或者说域名：blog.csdn.net
参数列表：spm=1018.2226.3001.5343

```


ArrayList 为什么比LinkList 查询快
```
ArrayList 底层用的数组，数组在内存中的地址是连续相邻的。
数组的查询，实际上是对引用地址的访问，不需要遍历。

ArrayList 为什么增删慢？
1、每插入和删除时，对应的需要向前或者向后元素。
2、当插入元素，判断是否需要扩容
3、扩容的操作，就需要重新创建一个新数组，再将元素放进去，较为繁琐。
```


0x7fffffff的意思
```
它的值是： 2147483647  （int 的最大值）
"0"代表正数
“x”则代表十六进制
```

"#ff0000"
```
使用 # 的符号来表示而非用个别的符号表示十六进制。24-bit 颜色可以用 #RRGGBB
的格式来表示，RR 是颜色中红色成分的数值，GG 是颜色中绿色成分的数值，BB
颜色中蓝色成分的数值。

00 是完全透明。 FF 是完全不透明
```

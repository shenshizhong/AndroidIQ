1、mac 环境的安装
```
进到一个目录
将下载的zip包放入，并解压

cd ~/development
unzip flutter_macos_v1.9.1+hotfix.6-stable.zip  

export PATH="$PATH:`pwd`/flutter/bin"
flutter precache
```


水仙花数的实现。
```
个位数的3次方，加上十位数的3次方，加上百位数的3次方，会等于本身，那么这个数就是水仙花数。

怎么获取个位数： 对数字取模于10，可以得到个位数。
怎么获取十位数： 先整除于10，得到两位数，再取模于10，就可以得到十位数了。
怎么获取百位数： 直接整除于100，获取到百位。

  int num = 100;
    do {
        int a = 0;
        int b = 0;
        int c = 0;
        a = num % 10; //获取数字的个位
        b = num / 10 % 10; //获取数字的十位
        c = num / 100; //获取数字的个位
        if (a*a*a + b*b*b + c*c*c == num){
            cout << "获取水仙花数：" << num << endl;
        }
        num++;
    } while (num < 1000);
```

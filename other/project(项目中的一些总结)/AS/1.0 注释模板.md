
新建一个类，添加注释模板
```
AS-> Preferences -> Editor -> File and Code Templates
-> File Header 在这里面添加如下即可：

/**
* @author:${USER}
* @date: ${DATE}
*/

给方法的注释模板：
只需要输入/** +Enter 就可以自动生成
```

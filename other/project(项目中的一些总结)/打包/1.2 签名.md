获取签名：
```
最简单的方式：
点击工具 android studio 右上角的gradle

app -> Tasks -> android -> signingReport(双击即可)

注意：要选择app的Task，如果选择项目的Task，拿到的是debug包的签名
如果不确定，在日志中查看到，Config: release，这个就是正式包的签名。

```

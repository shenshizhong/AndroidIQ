

X5WebView 内存泄漏的解决方案
```
 public void onDestroy() {
        if (x5WebView != null) {
            x5WebView.stopLoading();
            x5WebView.removeAllViewsInLayout();
            x5WebView.removeAllViews();
            x5WebView.setWebViewClient(null);
            try {
                CookieSyncManager.getInstance().stopSync();
            } catch (Exception e) {
                e.printStackTrace();
            }
            x5WebView.destroy();
            x5WebView = null;
        }
       super.onDestroy();
}

备注：super.onDestroy()， 要放在最后
```

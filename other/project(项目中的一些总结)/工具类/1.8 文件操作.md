

java char '\u0000'
```
'\u0000' 是char 的最小值，它转换成整数是0。
'\u0000' 与 '' 是不相等的。

测试：
 char c;
 Character character;
打印出char c; c的结果：[]   (一个小方块)
打印出Character character; character的结果：null


实际的应用场景：
final boolean isInvalid() {
        if (status == null) {
            status = (this.path.indexOf('\u0000') < 0) ? PathStatus.CHECKED
                                                       : PathStatus.INVALID;
        }
        return status == PathStatus.INVALID;
    }
```


gradle依赖的统一管理
```
新建 versions.gradle 文件


def deps = [:]
deps.android_gradle_plugin = "com.android.tools.build:gradle:4.2.0-alpha02"
ext.deps = deps

使用：

在项目的buiild.gradle 中加入如下，
apply from: 'versions.gradle'
buildscript {
    apply from: 'versions.gradle'
    dependencies {
        classpath deps.android_gradle_plugin
    }
}

```

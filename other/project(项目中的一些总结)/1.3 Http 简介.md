
1、http 简介
```
1、超文本传输协议

2、基于TCP/IP通信协议来传输数据

3、一个属于应用层的面向对象的协议

4、客户端通过url 向服务端发起请求，服务端接收到请求后，向客户端响应信息。

```
2、http 请求方法
```
特点：
Http 允许传输任意类型的数据对象，通过Content-Type 标记

Host 指的是请求的目的地。

User-Agent 浏览器类型检测的逻辑的重要基础


分为四部分：
1、请求行（请求方式，Http版本）
2、请求头部
3、空行
4、请求体
```
3、response
```
1、状态行 （状态码，Http 版本）
2、消息报头
3、空行
4、响应的正文

```

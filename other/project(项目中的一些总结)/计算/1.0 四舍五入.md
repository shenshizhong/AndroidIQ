
四舍五入
```
double a = 35;
double b = 20;
double c = a/b; //等于1.75
System.out.println(Math.ceil(c)); //向上取整输出2.0
System.out.println(Math.floor(c));  //向下取整输出1.0
System.out.println(Math.round(c));  //四舍五入取整输出2.0
```

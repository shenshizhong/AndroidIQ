
DecimalFormat 中的 0 和 #
```
0 代表只能有几位
# 代表最多有几位

常用的0.00 或者0.##

DecimalFormat df1 = new DecimalFormat("0.00");
DecimalFormat df2 = new DecimalFormat("0.##");
System.out.println(df1.format(0.1)); // 结果是 0.10
System.out.println(df2.format(0.1)); // 结果是 0.1
```

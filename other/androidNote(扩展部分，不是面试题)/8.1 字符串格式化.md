2、常用占位符
```
%s    字符串类型
%d    整型
%n    用于换行


	    String str=String.format("Hi,%s", "王力");
	    System.out.println(str);
>>>	    Hi,王力
```

3、常用转换符号
```
&#160;  空格  比如： &#160;人参与
\n    换行
\t    缩进

"  ： &#34; 或 &quot; 
'  ： &#39; 或 &apos;
&  ：  &#38; 或 &amp;
```


1、lambda 作用可以理解成是方便的移动代码块
 ```
 -> 后面表示的是函数体
 函数体的最后一句是表达式的返回值
 ()用来调用表达式，相当于invoke方法
 Lambda 可以写在括号外面
 面向抽象的思维来理解，更好理解，相当于是用一个碗去装东西回来。
 ```


2、lambda 的几种形式
```
 () -> Int //无参，返回 Int  
 (Int, Int) -> String //两个整型参数，返回字符串类型 
 (()->Unit, Int) -> Unit //传入了一个 Lambda 表达式和一个整型，返回 Unit 
```
